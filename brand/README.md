# Gantry brand assets

This directory is the canonical home for everything related to the customer-facing
**Gantry** brand. The agile-cards-board product ships under this brand identity.

## Files

| File | Purpose |
|---|---|
| `BRAND_HANDOFF_GANTRY_2026-06-18.md` | Source of truth for the brand identity. Generated by Gemini Pro, finalized 2026-06-18. |
| `tokens.css` | Brand primitives exposed as CSS custom properties. Import once in your app's globals. |
| `tailwind.preset.cjs` | Tailwind preset exposing the same primitives as utility classes (`gantry-forest`, `font-display`, etc.). Extend your `tailwind.config` with this. |
| `gantry-logotype-1.png` | Logotype reference, primary lockup. |
| `gantry-logotype-2.png` | Logotype reference, social/og:image variant. |
| `gantry-motion-1.mp4` | Motion reference, card drop animation. |
| `gantry-motion-2.mp4` | Motion reference, blueprint motion. |

## Quick usage

In a Tailwind app:

```js
// tailwind.config.cjs
module.exports = {
  presets: [require("../brand/tailwind.preset.cjs")],
  content: ["./index.html", "./src/**/*.{ts,tsx,jsx,js}"],
};
```

In the app's main stylesheet (alongside `@tailwind` directives):

```css
@import "../../brand/tokens.css";
```

For the wordmark in markup:

```jsx
<span className="font-mono font-bold tracking-logotype lowercase text-gantry-gunmetal">
  gantry
</span>
```

For the active-state lockup (on the main board route):

```jsx
<span className="font-mono font-bold tracking-logotype lowercase text-gantry-gunmetal">
  gantry
</span>
<span className="font-mono font-bold text-gantry-forest" aria-hidden="true">
  [ &#9646; ]
</span>
```
